HTML

  • You should declare the html document at the top with <!DOCTYPE_html>
  • The head element contains information and metadata about the webpage, and is not normally visible to users.
  • The head may contain meta tags as well as <title>, <style>, <base>, <link>, <script>, etc.
  • The body element represents the content the user will be shown.
  • Inside the body element are usually a header <h#> </h#> where # can be 1-6 and should be only nested in order. This is important for accessibility in screen readers and SEO (search engine optimization)
  • You can uniquely identify elements, like headers, ex. <header id="name">
  • As a side note, you can show html element formats in text form by showing the <, element name, and > putting a <q> before the element name and </q> after it inside the <> brackets without the quotation marks
